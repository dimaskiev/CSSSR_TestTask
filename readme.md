# CSSSR Project Test Task
### Тестовое задание в компанию CSSSR
**Для быстрого старта**


### Клонировать репозиторий и перейди в папку проекта
```
git clone git@github.com:dimaskiev/CSSSR_TestTask.git new-project && cd new-project
```

### Установить модули


```
yarn install
```

### Запустить шаблон
```
npm start
```

### Production
```
npm run production
```


### Deploy
```
npm run deploy
```
